Mein erstes automatisiertes Docker Deployment

Diese Webseite wurde mit GitHub Actions auf eine EC2-Maschine deployed.

Team: Baris

Deployment Schritte

  1. index.html und Dockerfile im Ordner frontend/ erstellt
  2. Dockerfile: nginx:alpine als Base Image, index.html wird in den nginx-Webordner kopiert
  3. GitHub Actions Workflow erstellt: wird bei jedem Push auf main ausgelöst
  4. SSH-Verbindung zur EC2-Instanz wird aufgebaut
  5. Docker wird auf EC2 installiert (falls nicht vorhanden)
  6. Der Ordner frontend/ wird per SCP auf EC2 kopiert
  7. Docker Image wird auf EC2 gebaut: docker build -t frontend-image
  8. Alter Container wird gestoppt und gelöscht
  9. Neuer Container wird gestartet: Port 80 wird freigegeben
